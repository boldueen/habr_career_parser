# Installation

```bash
cd your/code/folder
git clone https://github.com/boldueen/medianation_habr_parser
cd medianation_habr_parser
python3.11 -m venv venv
. venv/bin/activate
```

## install dependencies
```pip
pip install -r requirements.txt
```

## create .env file 
```bash
touch .env
```
then configure it...

## run the parser
```venv
python parser.py
```
